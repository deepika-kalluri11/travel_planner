AI-Powered Travel Planner
Overview
The AI-Powered Travel Planner is a web-based application that assists users in planning their travel by providing optimal transportation options. By leveraging LangChain and Google GenAI, this tool intelligently processes user inputs and generates travel suggestions, including cabs, trains, buses, and flights, along with their estimated costs. The application aims to provide users with a seamless and efficient way to plan their trips with real-time recommendations.

Features
Multiple Travel Options: Offers recommendations for various travel modes like cabs, trains, buses, and flights.
AI-powered Suggestions: Uses Google GenAI for intelligent response generation and LangChain to manage user queries in natural language.
Cost Estimations: Provides estimated prices for each travel mode, helping users compare costs.
User-friendly Interface: Built with Streamlit, ensuring a smooth and intuitive user experience.
Tech Stack
Programming Language: Python
User Interface: Streamlit
Framework: LangChain
AI Model: Google GenAI
How It Works
User Input: The user enters their source and destination locations.
Data Processing: The system processes the input using LangChain and generates queries for Google GenAI.
Recommendation Generation: Google GenAI returns travel options based on real-time data, including cabs, trains, buses, and flights.
Cost Estimates: The application estimates and displays the cost for each travel option.
Response Display: The user receives structured travel suggestions with pricing and options.
